Verion 1.0 ( 15-May-2017)

This is a sample NODE JS application , using Express framework and REST API (Salesforce). Use the Password with your security token to avoide failures. Added only Account object for reference. You can create record using just Account Name and Phone number. There is no validations and no error handling as of now. If there are any errors you will not be able to see it on the UI.
